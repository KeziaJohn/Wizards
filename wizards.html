Introduction

- The project is about re-engineer database to data warehouse.
- Core expectation of this project is to de-normalize the database to be available for user for reporting and analytics requirement.
- Creating and populating source tables and export tables data into .csv files.
- Perform Extract, Transform, Load (ETL) process on the data using pandas .
- Creating target database using PostgreSQL.

Usecase

- For Reporting and analysis
- For predictive analytics
- For Data Mining

Tech Stack Used

- MySQL
- PostgreSQL
